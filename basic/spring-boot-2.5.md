##  新特性

**Boot目标**： 自动化

**2.x 新特性总结**

- Java 8 起步
- Tomcat >= 8.5 Jetty >= 9.4 Hibernate >= 5.2
- 增加Reactive响应式模块
- HTTP/2 协议支持
- 采用 HikariCP 代替 Tomcat内置连接池 
- 支持Kotlin 1.2，支持性能监控Micrometer集成Actuator
